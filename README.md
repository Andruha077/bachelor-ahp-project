Моделювання системи вибору методом МАІ

Веб-застосунок для бакалаврської роботи на тему
«Розробка системи підтримки прийняття рішень на основі методу аналізу ієрархій (МАІ)»

📌 Про проєкт

Даний проєкт реалізує демонстраційну систему підтримки прийняття рішень із використанням методу аналізу ієрархій (МАІ).

Метою роботи є моделювання процесу оптимального вибору  на основі багатокритеріального аналізу.

Система дозволяє:

задавати важливість критеріїв;

формувати матрицю попарних порівнянь;

перевіряти узгодженість суджень (CR);

обчислювати вагові коефіцієнти;

визначати оптимальну альтернативу.

Проєкт демонструє синергію інтерфейсної частини та математичного апарату МАІ.

🛠 Використані технології

HTML5 – структура сторінки

CSS3 – стилізація та адаптивний дизайн

JavaScript – реалізація алгоритмів МАІ та обчислень

Розрахунок вагових коефіцієнтів

Перевірка узгодженості (CI, CR)

Візуалізація інтегральних пріоритетів

Визначення оптимального рішення
