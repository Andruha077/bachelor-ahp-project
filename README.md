«Розробка системи, що підтримує прийняття рішень на основі методу аналізу ієрархій »

📌 Про проєкт

Даний проєкт реалізує демонстраційну систему підтримки прийняття рішень із використанням методу аналізу ієрархій (МАІ).

Метою роботи є моделювання процесу оптимального вибору на основі багатокритеріального аналізу.

Система дозволяє:

задавати важливість критеріїв;

формувати матрицю попарних порівнянь;

перевіряти узгодженість суджень (CR);

оцінювати вагові коефіцієнти;

визначати оптимальну альтернативу.

Проєкт демонструє поєднання інтерфейсної частини та математичного апарату методу МАІ.

🛠 Використані технології

HTML5 – структура сторінки

CSS3 – стилізація та адаптивний дизайн

JavaScript – реалізація алгоритмів МАІ та обчислень

📂 Структура проєкту
index.html   – головний файл застосунку (інтерфейс та логіка МАІ)
README.md    – опис проєкту та його призначення
LICENSE      – ліцензія на використання коду
.gitignore   – файл виключення службових файлів з репозиторію
⚙️ Основний функціонал

Панель користувача для введення суб'єктивних оцінок

Автоматичне формування матриці попарних порівнянь

Розрахунок вагових коефіцієнтів

Перевірка узгодженості (CI, CR)

Візуалізація інтегральних переваг

Визначення оптимального рішення

👨‍💻 Інструкція для розробника

Нижче наведено покрокову інструкцію для запуску проєкту на новому комп'ютері.

1. Необхідне програмне забезпечення

Перед початком роботи необхідно встановити:

Git

Visual Studio Code (або інший редактор коду)

сучасний веб-браузер (Google Chrome, Firefox або Edge)

2. Клонування репозиторію

Відкрити термінал та виконати команду:

git clone https://github.com/username/project-name

Після цього перейти у папку проєкту:

cd project-name
3. Налаштування середовища розробки

Відкрити папку проєкту у редакторі коду (наприклад Visual Studio Code).

Рекомендується встановити розширення:

Live Server

4. Встановлення залежностей

Даний проєкт використовує стандартні веб-технології:

HTML

CSS

JavaScript

Тому додаткові залежності встановлювати не потрібно.

5. Налаштування бази даних

У даному проєкті база даних не використовується, оскільки всі обчислення виконуються на стороні клієнта.

6. Запуск проєкту у режимі розробки
Спосіб 1 (простий)

Відкрити файл

index.html

у браузері.

Спосіб 2 (через Live Server)

У Visual Studio Code натиснути правою кнопкою на файл index.html та обрати:

Open with Live Server
7. Базові команди Git

Отримати оновлення з репозиторію:

git pull

Додати зміни:

git add .

Створити коміт:

git commit -m "опис змін"

Відправити зміни:

git push
